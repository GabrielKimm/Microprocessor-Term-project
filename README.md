# Microprocessor Term-project

## Purpose & Scope
마이크로 프로세서 텀프로젝트의 공기청정기 제작을 목적으로 한다.

사용부품 목록

*  GP2Y1010AU0F 미세먼지 센서
*  PPD42NS 미세먼지 센서
*  Z80 processor 보드
*  L298N Motor driver
*   5v DC Motor
*   5 x 7 Dot Matrix Display
*   4 x 4 Switch Keypad
*   Air compression blade
